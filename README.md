# andcards-icons-kit

andCards Icons Kit for React and React Native.

![](iconset-preview.png)

### The Gist

```javascript
import Apple from "andcards-icons-kit/Apple";

<Apple color="grey" size={32} />
```

## API

| prop    | type     | default value |
| ------- | -------- | ------------- |
| `color` | `string` | #000000       |
| `size`  | `number` | '20'          |
